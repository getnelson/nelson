Welcome to Nelson

Please take a moment to install the command line tool which is the primary way to interact with Nelson.

curl -GqL http://bit.ly/2uDSQ7Y | bash

It is safe to rerun this script to keep nelson-cli current. Before getting started, ensure that you have followed these instructions the first time you install:

  1. Obtain a Github personal access token
  2. Add the Github token to your environment: export GITHUB_TOKEN=XXXXXXXXXXXXXXXX you may wish to add this to your ~/.bashrc for on-going convenience.
  3. Login to the instance: nelson login nelson.yourcompany.com
  4. Now you're ready to start using the Nelson!

Once logged in, you can use the built-in --help on every subcommand to see exactly what arguments are needed. Please also take some time to read the documentation and familiarize yourself with the core system concepts.

Thank you for using Nelson.
